# Issue & Sprint Management System

This project contains the complete implementation of the Issue and Sprint Management System, including a FastAPI backend and a React frontend.

## Project strucutre

- backend/: FastAPI application and API tests
- frontend/: React + Vite client application

## Branch strategy

- main: final complete project will be pushed
- python/dev: feature wise integration through pull request approvals
- python/feature-*: feature branches for each feature (major)
